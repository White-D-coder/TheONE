# Engineer OS: AI Review & Distraction Control

## AI Review Engine (The Strict Mentor)
The AI doesn't just track data; it analyzes it with a sharp, honest tone.
- **Daily Review**: "You planned 4 hours of Deep Work but spent 2 hours on Twitter. Tomorrow, start with a Focus Lock."
- **Weekly Review**: "Technical growth is stalling. You're picking easy tasks. Increase difficulty."
- **Monthly Strategy**: AI suggests role weight adjustments based on actual performance.

## Distraction Control Engine
Preventing "Drift" before it happens.
### Features:
- **Drift Detection**: If a user logs "Distraction" three times in a block, the OS triggers a "Hard Reset" (5 min walk, no screens).
- **Focus Protection**: During deep work blocks, the UI collapses to show only the active task and a timer.
- **Anti-Planning Trap**: If a user spends more than 30 mins planning without starting, the OS forces a "Minimum Task" (just 10 mins of execution).

## Tone and Character
The AI is:
- **Direct**: No sugarcoating.
- **Action-Oriented**: Always suggests a fix for a failure.
- **Evidence-Obsessed**: Doesn't care about "feeling productive".
